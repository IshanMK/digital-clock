# digital-clock

command line based digital clock created with C

## Features

- 24 hour format
- multiple colors

## compile

```bash
gcc -Wall -o E18173project1 E18173project1.c
```

## get the help menu

```bash
./E18173project1 -h
```

## set the colors for the clock

```bash
./E18173project1 -c red
```
